Data Format Guide

Upload a plain-text file (.csv, .txt, or .tsv) containing a univariate time series.

Accepted layouts

Single column (one value per line)

0.523
0.891
1.204
0.667
...

Two columns: index/time, value

0, 0.523
1, 0.891
2, 1.204
3, 0.667
...

Separated by comma, semicolon, tab, or space. The first column is ignored; only the last column is used as the time series.

Rules

  • A header row (non-numeric first line) is automatically skipped.
  • Empty lines and whitespace-only lines are ignored.
  • All values must be finite numbers.
  • For β-divergence, all values must be strictly positive (> 0).
  • Max recommended length: ~2000 samples (browser computation).
